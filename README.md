# create-docker

```
yarn create docker --node 6
```

## Usage

```
Options:
  --out, -o  The path to your Dockerfile, e.g. Dockerfile.dev
                                         [default: "./Dockerfile"]
  --node     The version of Node to run in Docker
                                               [default: "latest"]
  --env      The NODE_ENV value: e.g. production, staging,
             development                  [default: "development"]
  --help     Show help                                   [boolean]
```
